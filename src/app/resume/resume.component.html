Proyectos personales


Web Service CRUD Application.

• Creación de aplicaciones en Java que exponen APIs RESTful utilizando Spring Boot e integrando Maven para la gestión de dependencias y ciclo de vida del proyecto. Las aplicaciones interactúan con una base de datos MySQL cuyo esquema es gestionado y versionado mediante Liquibase. Para garantizar la portabilidad y consistencia del entorno, he utilizado Docker para crear contenedores, incluyendo un contenedor con la base de datos MySQL y otro con la aplicación empaquetada en un archivo JAR. Los proyectos están completamente automatizados a través de Dockerfile, permitiendo una fácil creación y despliegue de los contenedores. Además, he probado las APIs utilizando Postman, asegurando su correcto funcionamiento en distintos escenarios.
